About This Project

Quick Start

1. Installation

# Clone the repository
git clone https://github.com/vonhoff/wcag-review-demo.git
cd wcag-review-demo

# Install dependencies
pip install -r requirements.txt

2. Configuration

Copy the example environment file and add your credentials:

cp .env.example .env
# Edit .env with your GitHub token and Anthropic API key

3. Usage

# Analyze a PR
python -m src.main <pr_number>

# Or set PR number in environment
export GITHUB_PR_NUMBER=123
python -m src.main

Analysis Types

The tool supports multiple analysis types that can be configured via the ANALYSIS_TYPE environment variable:
  • review - Comprehensive code review with feedback on quality and best practices
  • summary - Concise summary of changes and their significance
  • security - Security vulnerability analysis and recommendations
  • accessibility - WCAG compliance check and accessibility improvements

Architecture

The project follows clean architecture principles with clear separation of concerns:

  • GitHubDiffFetcher - Handles all GitHub API interactions
  • DiffFilter - Filters diffs based on configurable criteria
  • ClaudeClient - Manages communication with Anthropic Claude API
  • PRDiffAnalyzer - Orchestrates the complete workflow

Testing

# Run all tests
pytest

# Run with coverage
pytest --cov=src --cov-report=html

# Run specific test file
pytest tests/test_github_diff_fetcher.py

GitHub Actions Workflow

The utility can be triggered via a manual GitHub Actions workflow, allowing you to analyze PRs directly from the GitHub UI. The workflow is configured to accept the PR number as input.
